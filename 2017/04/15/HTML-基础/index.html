HTML基础

2017-04-15 • 标签: HTML, 基础

1. HTML 简介

  • HTML 是用来描述网页的一种语言。
  • 指的是超文本标记语言 (Hyper Text Markup Language)
  • 不是一种编程语言，而是一种标记语言 (markup language) ，即一套标记标签(markup tag)
  • 使用标记标签来描述网页
  • HTML 文档 = 网页(描述网页)
  • HTML 文档包含 HTML 标签和纯文本
  • HTML 文档是由 HTML 元素定义的
  • Web浏览器的作用是读取 HTML 文档
  • HTML 参考手册

2. HTML 元素

开始标签 元素内容 结束标签
<p> This is a paragraph </p>
<a href=”default.htm” > This is a link </a>
<br /> 某些 HTML 元素具有空内容（空元素在开始标签中关闭） ＆nbsp;

3. HTML 块级元素

从新行开始，可容纳内联元素和其他块元素
元素的高度、宽度、行高以及顶和底边距都可设置
默认状态下块状元素的宽度都为100%,即在不设置宽度的情况下，是它本身父容器的100%（和父元素的宽度一致），除非设定一个宽度
将内联元素变块级元素—display:block

  • <address> - 地址
  • <blockquote> - 块引用
  • <center> - 举中对齐块
  • <dir> - 目录列表
  • <div> - 常用块级容器,也是csslayout(布局)的主要标签。<div>可定义文档中的分区或节，可以把文档分割为独立的、不同的部分。对同一个 <div> 元素应用 class 或 id属性，可实现对大的内容块应用样式
  • <dl> - 定义列表
  • <fieldset >- form控制组
  • <form> - 交互表单（只能用来容纳其它块元素）
  • <h1> - <h6>
  • <hr >- 水平分隔线
  • <isindex> - input prompt
  • <menu> - 菜单列表
  • <noframes> - frames可选内容，（对于不支持frame的浏览器显示此区块内容
  • <noscript> - 可选脚本内容（对于不支持script的浏览器显示此内容）
  • <ol >- 排序表单
  • <p> - 段落
  • <pre> - 格式化文本 (可用于显示多行代码)
  • <table> - 表格
  • <ul> - 非排序列表

3. HTML 内联元素(行内元素)

不从新行开始，只能容纳文本或者其他内联元素
和其他元素都在一行上
元素的高度、宽度、上下边距不可设置(设置了也无效)
元素的宽度就是它包含的文字或图片的宽度，不可改变
将块级元素变内联元素—display:inline

内联元素之间有一个间距bug问题:当行内元素之间有“回车”、“tab”、“空格”时就会出现间隙，
解决方法：1、写在一行，之间不要有空格之类的符号; 2、使用font-size:0

  • <big >- 大字体
  • <br >- 换行
  • <cite> - 引用

  • <dfn> - 定义字段

  • <font> - 字体设定(不推荐)

  • <label> - 表格标签
  • <q> - 短引用
  • <s> - 中划线(不推荐)
  • <samp>- 定义范例计算机代码
  • <select>- 项目选择
  • <small>- 小字体文本
  • <span>- 常用内联文本容器，定义文本内区块。用来组合文档中的行内元素，没有固定的格式表现，如不对span应用样式，那么span元素中的文本与其他文本不会有任何视觉上的差异。当对它应用样式时，才会产生视觉上的变化

6. HTML 标签

  • 即由尖括号包围的关键词，比如 <html>
  • 通常是成对出现的，比如 <b>开始标签(开放标签) 和</b>结束标签(闭合标签)
  • <html> 与 </html> 之间的文本描述网页(<html>为文档的根元素)
  • <head> 与 </head>之间定义文档的头部，可用在head部分的头部元素:<base>, <link>, <meta>,<script>, <style>,<title>
  • <body> 与 </body> 之间的文本是可见的页面内容(主体)
  • <!DOCTYPE>声明—帮助浏览器正确地显示网页（<!DOCTYPE html>）

7. HTML 属性

  • 总是以名称/值对的形式出现，比如：name=”value”
  • 始终为属性值加引号(个别情况下属性值本身就含有双引号，则必须使用单引号)
  • 适用于大多数 HTML元素的属性：
    • class–规定元素的类名（classname）；
    • Id–规定元素的唯一id；
    • style–规定元素的行内样式（inline style）；
    • Title—规定元素的额外信息（可在工具提示中显示）.

8. HTML 样式

  • style 属性: 提供了一种改变所有 HTML 元素的样式的通用方法

  • 三种方式来插入样式表：

    • 外部样式表
      当样式需要被应用到很多页面的时候，可使用外部样式表，在内部通过<link>标签定义外部样式表

    • 内部样式表
      当单个文件需要特别样式时，可使用内部样式表。在<head>内部通过 <style> 标签定义内部样式表。

    • 内联样式
      当特殊的样式需应用到个别元素时,可使用内联样式。即在相关的标签中使用样式属性，样式属性可以包含任何 CSS 属性

15. HTML 链接

  • <a href="http://www.baidu.com.cn">This is a link</a>
  • 使用 href 属性 - 创建指向另一个文档的链接

  • 使用 name 属性 - 创建文档内的书签，规定锚（anchor）的名称，实现文档内的跳转

  • 使用 Target 属性 - 定义被链接的文档在何处显示

    • _blank - 在新窗口中打开被链接文档。
    • _self（默认）- 在相同的框架中打开被链接文档。
    • _parent- 在父框架集中打开被链接文档。
    • _top- 在整个窗口中打开被链接文档。
    • framename- 在指定的框架中打开被链接文档。

17. HTML 表格

  • 由 <table> 标签来定义，如<table border="1" cellpadding="10" cellspacing="10" >
    （cellpadding创建单元格内容与其边框之间的空白，cellspacing创建单元格之间的距离）
  • 每个表格均有若干行（<tr>），每行被分割为若干单元格（<td>）
    包括表格数据，即数据单元格的内容
  • 表格的表头使用<th>标签进行定义（相当于<td>）
  • 如某个单元格是没内容，在空单元格中添加一个空格占位符&nbsp;，就可将边框显示出来
  • 属性：colspan=”2”—-占两列；rowspan=”2”—-占两行；align—排列单元格内容
  • <caption>表格标题</caption>
  • 用<tbody>..</tbody>可以使表格在表格内容全部下载后才显示
  • 为表格添加边框一般CSS语法：table tr td,th{border:1px solid #000;}

18. HTML 列表

  • 无序列表

    • 始于 <ul>标签，每个列表项始于<li>
    • 浏览器使用粗体圆点进行标记

  • 有序列表

    • 始于 <ol>标签，每个列表项始于<li>
    • 浏览器使用数字进行标记

  • 自定义列表

    • 以 <dl>标签开始，每个自定义列表项以<dt>开始，其内容定义以 <dd> 开始

    • 可实现嵌套

    • type属性可定义列表标记方式

24. HTML 长度值

  1. 像素
  • 像素指的是显示器上的小点（CSS规范中假设“90像素=1英寸”）
  • 实际情况是浏览器会使用显示器的实际像素值有关，在目前大多数的设计者都倾向于使用像素（px）作为单位
  1. em
  • em就是本元素给定字体的font-size值，如元素的font-size为14px，那么1em=14px；如font-size为18px，那么1em=18px
  • p{font-size:12px;text-indent:2em;} //实现段落首行缩进 24px（也就是两个字体大小的距离）
  • 注意：当给 font-size 设置单位为 em 时，计算的标准以 p 的父元素的 font-size 为基础
  1. 百分比
  • p{font-size:12px; line-height:130%} //设置行高（行间距）为字体的130%（12 * 1.3 = 15.6px）

25. HTML 表单

  • 用于搜集不同类型的用户输入
  • <form>元素
    • action 属性定义在提交表单时执行的动作(向何处提交表单)
    • method属性规定在提交表单时所用的 HTTP 方法（GET 或 POST）。

      GET（默认方法）：表单提交是被动的（比如搜索引擎查询），没有敏感信息，表单数据在页面地址栏中是可见的，最适合少量数据的提交;
      POST：表单正在更新数据或者包含敏感信息（例如密码）时使用，在页面地址栏中被提交的数据是不可见的，更加安全。

    • target属性规定在何处打开action属性定义的URL
    • <form>包含的是表单元素，表单元素指的是不同类型的 input 元素、复选框、单选按钮、提交按钮等等
  • 表单元素<input>：
    • <input type="text"> 定义用于文本输入的单行输入字段
    • <input type="radio"> 定义单选按钮
    • <input type="checkbox">定义复选框
    • <inputtype="submit">定义用于向表单处理程序提交表单的按钮
      (表单处理程序通常包含用来处理输入数据的脚本的服务器页面，在
      的action属性中指定)
    • <input type="password"> 定义密码字段
    • <input type="button> 定义按钮
    • HTML5增加了多个新的type值:color,date,datetime,datetime-local,email,month,number,range,search,tel,time,url,week
    • 如果要正确地被提交，每个输入字段必须设置一个name属性
    • 其他属性：
      disabled 规定输入字段应该被禁用。
      max 规定输入字段的最大值。
      maxlength 规定输入字段的最大字符数。
      min 规定输入字段的最小值。
      pattern 规定通过其检查输入值的正则表达式。
      readonly 规定输入字段为只读（无法修改）。
      required 规定输入字段是必需的（必需填写）。
      size 规定输入字段的宽度（以字符计）。
      step 规定输入字段的合法数字间隔。
      value 规定输入字段的默认值。
    • HTML5新增属性：
      autocomplete 规定表单或输入字段是否应该自动完成
      autofocus
      form 规定<input>元素所属的一个或多个表单
      (以下几个适用于 type=”submit” 以及 type=”image”)
      formaction 规定当提交表单时处理该输入控件的文件的 URL
      formenctype 规定当把表单数据提交至服务器时如何对其进行编码（仅针对 method=”post” 的表单）
      formmethod 定义用以向 action URL 发送表单数据的HTTP方法
      formnovalidate 规定在提交表单时不对 <input>元素进行验证
      formtarget 规定的名称或关键词指示提交表单后在何处显示接收到的响应
      height 和 width
      list
      min 和 max
      multiple
      pattern 规定用于检查 <input>元素值的正则表达式
      placeholder
      required
      step

  • 表单元素<select>：

  • 表单元素<textarea>：
