CSS-布局

2017-04-24 • 标签: CSS

  • 布局模型是建立在盒模型的基础之上

  • 清除默认样式(初始化页面)：
    body{margin:0; padding:0} //一般设置布局都要使用

  • 设置top、right、bottom、left时偏移的参照物为其已定位(一般是相对定位)的父元素

  • 先考虑设计图中的文字内容和内容模块之间的关系，重点放在编写html结构和语义化，然后考虑布局和表现形式

  • 最大化的简化html的结构，然后用css进行设置，减少html与css的契合度

1. 一列居中布局

  • 实现步骤：
    1. 设置水平居中:{margin: 0 auto;} (上下值任意，但左右值必为auto)
    2. 设置固定宽度：设置width为某个定值，但不能为100%(100%就不居中了)
    3. 高度自定义设置
  • 包含子模块的情况：用一个包裹层包住所有子层，各个子层级的宽度根据需求设置相对父层宽度的百分比(一般设为100%)，最后对包裹层进行上述设置
  • 典型一列布局：三个<div>：头部top、主体main、尾部foot (根据各模块的需要来设置居不居中以及宽度大小)
  • 注意：一旦设置 float属性或 position：absolute/fixed 则取消居中了
  • auto作用：根据浏览器宽度自动设置两边的外边距
  • 通常用于作首页
  • 不适用于存放文本

2. 横向两列布局

  • 宽度自适应的(较少使用)—————–使用浮动定位
    1. 设置两列宽度：使用%(因为要自适应)
    2. 需左右排列，所以分别设置float:left/right(因浮动而影响位置的元素可根据需求消除浮动)
    3. 高度自定义设置
  • 固定宽度的(一般是居中的)—————使用浮动定位
    1. 设置两列宽度：px(精确控制)/%(粗略控制)
    2. 需左右排列，所以分别设置float:left/right(因浮动而影响位置的元素可根据需求消除浮动)
    3. 高度自定义设置
    4. 然后用一个父div包住两列
    5. 设置父div的width，再设置{margin:0 auto;}
  • 一列固定宽度另一列自适应(较少使用)—–使用绝对定位
    1. 设置两列宽度：px(固定宽度)/%(自适应宽度)
    2. 自适应宽度元素绝对定位，其父元素相对定位(即设置绝对定位的元素一定在设置相对定位元素的里面)
    3. 设置偏移量：自适应宽度元素以其(相对)定位的父元素为参照基准进行偏移
      (因为已脱离标准文档流，所以一般需设置margin值，从而与固定宽度列有一定间距，不覆盖)
      (margin值一般为 固定宽度列宽度+间距 ，方向视情况而定)
    4. 注意：固定宽度列的高度 > 自适应宽度列的高度(利用固定宽度页面撑开父元素的页面)

3. 横向三列布局

  • 总宽度自适应的(较少使用)————使用浮动定位
    1. 设置列宽度：使用%(因为要自适应)
    2. 根据需求排列各列：分别设置float:left/right
    3. 高度自定义设置
  • 宽度有固定宽度也有自适应的———-使用绝对定位
    1. 左右列固定宽度，中间列宽度自适应
    2. 左列绝对定位：{position:absolute; left:0; top:0;}
    3. 右列绝对定位：{position:absolute; right:0; top:0;}
    4. 中间列去掉宽度定义，加上左右两侧margin值：{margin:0 310px 0 210px;}（假设右列300px，左列200px，剩下的10px是通过增加margin属性值实现列间空隙）
    5. 设置父div的width，再设置{margin:0 auto;}
    6. 注意：固定宽度列的高度 > 自适应宽度列的高度(利用固定宽度页面撑开父元素的页面)
