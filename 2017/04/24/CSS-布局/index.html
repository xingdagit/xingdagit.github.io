CSS-布局

2017-04-24 • 标签: CSS

  • 布局模型是建立在盒模型的基础之上

  • 清除默认样式(初始化页面)：body{margin:0; padding:0} //一般设置布局都要使用

1. 一列布局

  • 实现步骤：(居中型)
    1. 设置水平居中{margin: 0 auto;} (上下值任意，但左右值必为auto)
    2. 设置固定宽度：设置width为某个定值，但不能为100%(100%就不居中了)
    3. 高度自定义设置
  • 典型一列布局：三个<div>：头部top、主体main、尾部foot (根据各模块的需要来设置居不居中)
  • 若设置 float属性或 position：absolute/fixed 则取消居中了
  • auto作用：根据浏览器宽度自动设置两边的外边距
  • 通常用于作首页
  • 不适用于存放文本

2. 两列布局

  • 总宽度自适应的(较少使用)
    1. 设置两列宽度：使用%(因为要自适应)
    2. 需左右排列，所以分别设置float:left/right
    3. 高度自定义设置
  • 固定宽度的(一般是居中的)
    1. 设置两列宽度，px(精确控制)/%(粗略控制)
    2. 需左右排列，所以分别设置float:left/right
    3. 高度自定义设置
    4. 然后用一个父div包住两列
    5. 设置父div的width，再设置margin:0 auto;

3. 三列布局

  • 总宽度自适应的(较少使用)
    1. 设置列宽度：使用%(因为要自适应)
    2. 根据需求排列各列：分别设置float:left/right
    3. 高度自定义设置
  • 宽度有固定也有自适应的
    1. 左右列固定宽度，中间列宽度自适应
    2. 左列绝对定位：position:absolute; left:0; top:0;
    3. 右列绝对定位：position:absolute; right:0; top:0;
    4. 中间列去掉宽度定义，加上左右两侧margin值：margin:0 310px 0 210px;（假设右列300px，左列200px，剩下的10px是通过增加margin属性值实现列间空隙）
    5. 设置父div的width，再设置margin:0 auto;
    6. 高度自定义设置
