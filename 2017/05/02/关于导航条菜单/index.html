关于导航条菜单

2017-05-02

1. 垂直菜单的制作

  • 一般用ul无序列列表构建菜单
  • 清除默认样式 *{margin:0;padding:0;}
  • 清除ul的样式 ul{list-style：none} /去掉前面的默认小圆点/
  • 去掉<a>的下划线 a{text-decoration：none} /在导航栏中经常用/
  • 将行内元素<a>变为块状元素，从而可以设置 宽度和高度：ul li a{display:block;}
  • 根据需要设置<ul>的width、height、line-height
  • 如果要在文字前面增加距离，最好用文本缩进方式:a{text-indent：xx px}(最好不用padding，因为会增加总宽度)
  • 设置鼠标经过的文字背景颜色和字体颜色(交互效果)：a:hover{ background-color:#F60; color:#fff;}

2. 水平菜单的制作

  • 在垂直菜单设置的基础上设置
  • 对<li>设置浮动：li{float:left;} /将垂直菜单转变成水平菜单的关键/
  • 去掉<ul>的宽度的设置(若有)
  • 将文本缩进text-indent替换成{text-align:center;}
  • 如果要给水平菜单增加整体背景，需要给<ul>定义宽、高属性(因为菜单<li>浮动后脱离文档流，导致<ul>失去高度和宽度)

3. 圆角菜单的制作

  • 在水平菜单设置的基础上设置
  • 对<ul>设置height、border-bottom(用于装饰，颜色最好跟a:hover中背景颜色相同)
  • 此时水平菜单与下边框会有一段空隙，所以要用li{margin-top:**px;}使菜单往下移并与下边框接上
  • 根据需要用padding让菜单左右移动以达到美观
  • 实现圆角：border-radius属性
  • 选一个选项作为默认选中状态(一般为首页选项)：对相应<li>设置一个class，并将其与a:hover样式设置放在一起

4. 伸缩菜单的制作（高度改变）

  • 在圆角菜单设置的基础上设置
  • 去掉圆角属性设置
  • 增加高度值：设置选中状态(包括默认选中状态)的高度为较大点的高度。
  • 此时是选中选项是往下增加高度值的，所以会内容超出下边框。因此利用margin将其往上移
  • 因margin值的设置，文字位置也改变了。所以用line-height属性实现垂直居中
