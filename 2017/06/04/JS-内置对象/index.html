一、javascript 对象

  1. JavaScript 中的所有事物都是对象，如:字符串、数值、数组、函数等，每个对象带有属性和方法。
  2. JavaScript 提供多个内置对象，比如 String、Date、Array 等等。
  3. 使用对象前要先用new定义，用.调用其属性和方法。

  2. String 字符串对象

    • 定义：var mystr="Hello World!";
    • 访问属性：var myl=mystr.length;
    • 调用方法：var mynum=mystr.toUpperCase();
    • 一些方法：
      1)对象.toUpperCase();//将字符串转化成大写
      2)对象.toLowerCase();//将字符串转化成小写
      3)对象.charAt(index);//返回指定位置的字符(index：0~length-1，不在返回空字符串)(其实也可以用数组形式实现相同效果)
      4)对象.indexOf(需检索的字符串值,[开始检索位置]);// 返回某个指定的字符串值在字符串中首次出现的位置，没有返回-1
      5)对象.split(切割标志,[切割后元素数]); //将字符串分割为字符串数组，并返回此数组
      6)对象.substring(开始位置,[结束位置]);//提取字符串中介于[开始位置,结束位置-1)的字符
      7)对象.substr(开始位置,[提取字符串的长度]);// 从字符串中提取从开始位置起的指定数目(长度)的字符串(开始位置为负时表示从尾开始)

  3. Math对象

    • Math对象提供一些属性和方法对数据进行数学计算。
    • Math 对象是一个固有的对象，无需创建，直接把 Math 作为对象使用就可以调用其所有属性和方法。这是它与Date,String对象的区别。

    • 一些方法：
      1)Math.ceil (x);// 对一个数进行向上取整(返回的是大于或等于x，并且与x最接近的整数)
      2)Math. floor (x);// 对一个数进行向下取整(返回的是小于或等于x，并且与 x 最接近的整数)
      3) Math. round (x);// 把一个数字四舍五入为最接近的整数(返回与x最接近的整数)
      4) Math. random ();//返回一个大于或等于0但小于1的符号为正的数字值

  4. Array 数组对象

    • 数组对象是一个对象的集合，里边的对象可以是不同类型的。数组的每一个成员对象都有一个“下标”，用来表示它在数组中的位置，是从零开始的。
    • 数组定义的方法：
      1)var 数组名= new Array();//定义了一个空数组
      2)var 数组名 =new Array(n);// 定义时指定有n个空元素的数组
      3) var 数组名 = [<元素1>, <元素2>, <元素3>...];// 定义数组的时候，直接初始化数据
    • 数组元素使用：数组名[下标] = 值;//下标从0开始
    • 数组属性：
      length用法：<数组对象>.length；返回：数组的长度，即数组里有多少个元素。它等于数组里最后一个元素的下标加一。
    • 数组方法：

      1)对象.concat(array1,array2,...,arrayN);//用于连接两个或多个数组。此方法返回一个新数组，不改变原来的数组
      2)对象. join(分隔符);//用于把数组中的所有元素放入一个字符串并返回。元素是通过指定的分隔符进行分隔的
      3) 对象.reverse(); //用于颠倒数组中元素的顺序(该方法会改变原来的数组)
      4) 对象.slice(start,end); //从已有的数组中返回选定的元素(返回一个新的数组)
      5) 对象.sort([方法函数]);//使数组中的元素按照一定的顺序排列(不指定<方法函数>，则按unicode码顺序排列)
